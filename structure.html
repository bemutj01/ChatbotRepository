User Management System

Login

Enter your username and password to log in. (Python: loginPage.login() uses checkUserPass.)

Forgot password? Create an account

Registration

Create a new account. (Python: registrationPage.register() checks if the user exists, then creates a User.)

If the username already exists, the backend should print/return: "User already exists!"

Reset Password

Reset your password by entering your username and a new password. (Python: resetPassPage.resetPassword() looks up the user in users[].)

If the username is not found, backend response should match: "User not found."
